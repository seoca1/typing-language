## ⚙️ 기본 설정

### 권장 AI 도구

**무료:**
- Stable Diffusion WebUI (로컬)
- Bing Image Creator (DALL-E 3 기반)

**유료:**
- Midjourney (고품질)
- NovelAI (아니메 특화)
- DALL-E 3 (ChatGPT Plus)

### 공통 설정

**이미지 사양:**
- **크기:** 512×768 또는 768×1024
- **형식:** PNG (투명 배경 권장)
- **스타일:** 아니메/일러스트
- **화질:** High quality, detailed

**공통 네거티브 프롬프트:**
```
lowres, bad anatomy, bad hands, text, error, missing fingers, 
extra digit, fewer digits, cropped, worst quality, low quality, 
normal quality, jpeg artifacts, signature, watermark, username, 
blurry, artist name, multiple views, multiple angles, split screen
```

## 💡 생성 팁

### Stable Diffusion 설정

**모델:**
- Anything V5 (아니메 일반)
- AbyssOrangeMix (고품질 아니메)
- CounterfeitV3 (아니메 스타일)

**설정:**
```
Steps: 30-50
CFG Scale: 7-12
Sampler: DPM++ 2M Karras
Size: 512×768 or 768×1024
Denoising: 0.7
```

**VAE:**
- vae-ft-mse-840000-ema-pruned (색상 개선)

---

### Midjourney 팁

**버전:**
- `--v 6` 최신 버전 사용

**파라미터:**
```
--ar 2:3 (세로 비율)
--style raw (사실적)
--q 2 (고품질)
```

**예시:**
```
/imagine masterpiece anime girl with blonde hair, full body, 
standing pose, white background --ar 2:3 --v 6 --q 2
```

---

### NovelAI (아니메 특화)

**모델:**
- NAI Diffusion Anime V3

**설정:**
```
Resolution: Portrait (512×768)
Steps: 28
Scale: 11
SMEA: On
Quality Tags: masterpiece, best quality
```

---

### 배경 제거

**자동 배경 제거:**
- [remove.bg](https://remove.bg) - AI 자동 제거
- [PhotoScissors](https://photoscissors.com) - 온라인 도구
- GIMP - 무료 (Magic Wand 도구)

**수동 배경 제거 (고품질):**
1. Photoshop: Quick Selection → Refine Edge
2. GIMP: Select by Color → Feather → Delete
3. Krita: Selection Tools → Layer Mask

---

### 일관성 유지

**같은 캐릭터 여러 포즈:**

1. **시드 고정:**
   - Stable Diffusion: `--seed 12345`
   - Midjourney: `--seed 12345`

2. **LoRA 사용:**
   - 캐릭터 학습 후 재사용
   - 일관된 외모 유지

3. **프롬프트 템플릿:**
   ```
   [공통 외모] + [포즈] + [공통 설정]
   ```

4. **Img2Img:**
   - 첫 이미지 생성
   - Img2Img로 포즈만 변경
   - Denoising 0.5-0.7

---
