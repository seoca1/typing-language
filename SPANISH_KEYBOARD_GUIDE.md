# 🇪🇸 스페인어 키보드 설정 가이드

## 📌 개요

Typing Language의 스페인어 스테이지는 **2가지 입력 모드**를 지원합니다:

| 모드 | 표시 | 설명 | 키보드 요구사항 |
|------|------|------|-----------------|
| **Loose** | ⌨️ Loose (초록) | ASCII 폴백 허용 | ✅ 영어 키보드 가능 |
| **Strict** | ⌨️ Strict (주황) | 정확한 액센트 필요 | ⚠️ 스페인어 키보드 필요 |

---

## 🎮 게임 내 표시

### Loose 모드 (Tier 1)
```
┌──────────────────────────────────┐
│ ES Primeras Palabras ⌨️ Loose   │
│ 기본 어휘                        │
│ ✓ 영어 키보드 가능: a→á 자동인식│
└──────────────────────────────────┘
```

### Strict 모드 (Tier 2+)
```
┌──────────────────────────────────┐
│ ES Acentos (Strict) ⌨️ Strict   │
│ 액센트 단어                      │
│ ⚠️ 스페인어 키보드 필요: á é í... │
└──────────────────────────────────┘
```

---

## 🔤 스페인어 특수 문자

스페인어에서 입력해야 하는 특수 문자:

| 문자 | 이름 | 영어 폴백 (Loose 모드) |
|------|------|------------------------|
| **á** | a con tilde | a |
| **é** | e con tilde | e |
| **í** | i con tilde | i |
| **ó** | o con tilde | o |
| **ú** | u con tilde | u |
| **ñ** | eñe | n |
| **¿** | 거꾸로 물음표 | (없음) |
| **¡** | 거꾸로 느낌표 | (없음) |

## 💻 물리 키보드 설정

### 🍎 **macOS**

#### 방법 1: 스페인어 키보드 추가 (권장)
1. **시스템 설정** → **키보드** → **입력 소스**
2. **[+]** 클릭 → **스페인어** 선택
3. **Spanish - ISO** 또는 **Spanish** 선택
4. 추가 완료

**사용법:**
- 메뉴바 오른쪽 위 언어 아이콘 클릭
- **Spanish** 선택
- 액센트 문자가 키보드에 직접 매핑됨

**키 위치 (Spanish 키보드):**
```
á = [']  (세미콜론 위치)
é = [2]  (Shift + 2 대신 액센트)
í = [;]  
ó = [:]  
ú = [[] 
ñ = [;]  (영어 키보드의 세미콜론)
¿ = [-]  (Shift + -)
¡ = [1]  (Shift + 1)
```

#### 방법 2: Dead Keys (영어 키보드 유지)
1. **영어 키보드 그대로 사용**
2. 액센트 조합키 사용:

**사용법:**
```
Option + e, 그다음 a → á
Option + e, 그다음 e → é
Option + e, 그다음 i → í
Option + e, 그다음 o → ó
Option + e, 그다음 u → ú
Option + n, 그다음 n → ñ
Option + /, 그다음 Shift → ¿
Option + 1 → ¡
```

---

### 🪟 **Windows**

#### 방법 1: 스페인어 키보드 추가 (권장)
1. **설정** → **시간 및 언어** → **언어**
2. **언어 추가** → **Español (스페인)** 선택
3. **옵션** → **키보드** → **Spanish** 추가
4. 작업 표시줄 언어 표시기에서 전환

**사용법:**
- `Win + Space` 또는 작업 표시줄 언어 아이콘 클릭
- **ESP** (Spanish) 선택

**키 위치 (Spanish 키보드):**
```
á = [']
é = [2] (Shift 없이)
ñ = [;] (세미콜론 위치)
¿ = [-] + Shift
```

#### 방법 2: Alt 코드 (영어 키보드 유지)
1. **Num Lock** 켜기
2. **Alt** 누른 상태로 숫자 입력:

```
Alt + 160 → á
Alt + 130 → é
Alt + 161 → í
Alt + 162 → ó
Alt + 163 → ú
Alt + 164 → ñ
Alt + 168 → ¿
Alt + 173 → ¡
```

---

### 🐧 **Linux**

#### 방법 1: 스페인어 키보드 추가
1. **Settings** → **Region & Language**
2. **Input Sources** → **[+]**
3. **Spanish (Spain)** 선택

#### 방법 2: Compose Key (영어 키보드 유지)
1. **Settings** → **Keyboard** → **Compose Key** 설정
2. (보통 Right Alt로 설정)

**사용법:**
```
Compose + ' + a → á
Compose + ' + e → é
Compose + ~ + n → ñ
```

---

### 📱 **모바일 (iOS/Android)**

**자동 지원:**
- 모바일에서는 **가상 키보드**가 자동으로 표시됩니다
- 스페인어 스테이지 시작 시 액센트 키가 포함된 4번째 줄 표시:
  ```
  [á] [é] [í] [ó] [ú] [¿] [¡]
  ```
- 별도 설정 불필요

## 🎯 입력 모드 상세

### ✅ Loose 모드 (초급 - Tier 1)

**특징:**
- 영어 키보드로 플레이 가능
- 액센트 자동 인식
- 초보자 친화적

**예시:**
```
Target: "café"

✅ cafe  → 인정 (자동 정규화)
✅ café  → 인정
✅ CAFE  → 인정
✅ Café  → 인정
```

**적용 스테이지:**
- `es_1_1`: Primeras Palabras
- `es_1_2`: Saludos

---

### ⚠️ Strict 모드 (중급+ - Tier 2+)

**특징:**
- 스페인어 키보드 필요
- 정확한 액센트 입력 필수
- 실제 스페인어 학습

**예시:**
```
Target: "café"

❌ cafe  → 오답 (액센트 없음)
✅ café  → 정답
❌ CAFE  → 오답
❌ Café  → 오답 (대소문자 불일치)
```

**적용 스테이지:**
- `es_2_1`: Acentos (Strict) - Strict 모드 시작
- `es_2_2`: Cotidiano
- `es_3_1`: Frases Cortas
- `es_3_2`: Conversación

## 🔧 게임 내 지원 기능

### 1. **모드 표시**
- 스테이지 제목에 모드 뱃지 표시
- Loose: 초록색 ⌨️ Loose
- Strict: 주황색 ⌨️ Strict

### 2. **키보드 안내**
- 스테이지 설명 아래 안내 메시지
- Loose: "✓ 영어 키보드 가능"
- Strict: "⚠️ 스페인어 키보드 필요"

### 3. **가상 키보드 (모바일)**
- 액센트 키 자동 표시
- 터치로 직접 입력 가능

### 4. **기대 키 하이라이트**
- 다음 입력할 문자 초록색 펄스 표시
- 액센트 포함 문자도 정확히 표시

## ❓ FAQ

### Q1: Loose 모드에서도 액센트를 입력할 수 있나요?
**A:** 네! Loose 모드는 둘 다 인정합니다.
- `cafe` ✅ (영어 키보드)
- `café` ✅ (스페인어 키보드)

### Q2: Strict 모드에서 영어 키보드를 사용하면?
**A:** 오답 처리됩니다. Strict 모드는 정확한 액센트 입력을 요구합니다.

### Q3: 모바일에서는 어떻게 입력하나요?
**A:** 가상 키보드가 자동으로 액센트 키를 표시합니다. 별도 설정 불필요.

### Q4: ¿와 ¡은 언제 입력하나요?
**A:** 문장 스테이지 (Tier 3+)에서 등장합니다. 
- 물음문: `¿Cómo estás?`
- 감탄문: `¡Hola!`

### Q5: 키보드 전환이 번거로워요
**A:** 
- **Mac**: `Cmd + Space` 단축키로 빠른 전환
- **Windows**: `Win + Space` 단축키
- **추천**: Loose 모드로 먼저 익숙해진 후 Strict 도전

### Q6: 대소문자는 구분하나요?
**A:** Strict 모드에서는 **대소문자 구분**합니다.
- Target: `Hola` → 입력: `hola` ❌

## 🎓 학습 팁

### 1. **단계별 진행**
```
Tier 1 (Loose) → 기본 어휘 익히기
       ↓
키보드 설정 (10분 투자)
       ↓
Tier 2 (Strict) → 정확한 입력 연습
```

### 2. **손가락 위치 외우기**
스페인어 키보드에서 자주 쓰는 키:
- `ñ` = 오른손 새끼손가락 (세미콜론 위치)
- `á` = 오른손 새끼손가락 (따옴표 위치)

### 3. **Dead Keys 마스터 (Mac)**
```
Option + e = 액센트 준비
그다음 모음 = 액센트 완성
```
한 번 익히면 모든 프로그램에서 사용 가능!
