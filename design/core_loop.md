# Core Loop

게임의 매크로/미시 루프.

## 매크로 루프 (한 세션)

```
[메인 메뉴]
    ↓ 게임 시작
[스테이지 선택] — 언어 / 난이도 / 테마
    ↓ 시작
[스테이지 진행]
    ↓ (미션 클리어 또는 실패)
[스테이지 결과 화면] — WPM, 정확도, 점수, 미션 보상
    ↓
[메인 메뉴] — 메타 진행/해금 확인
```

## 미시 루프 (한 단어/문장 = 한 적)

```
[적 등장] — 화면 상단/측면에 단어 또는 문장
    ↓
[플레이어 타이핑] — 실시간 입력, 정확도 추적
    ↓
[입력 완료] — 적 처치, 다음 등장 또는 스테이지 종료
    ↓ (반복)
```

## 미시 루프의 두 가지 모드

### 모드 A: 격파 모드 (Combat / Defeat)
- 단어/문장이 적. 타이핑 완료 = 처치.
- HP 시스템(선택): 단어 길이 = HP, 오타 1개 = -10% 데미지
- 화면 진행: 적이 플레이어를 향해 다가옴, 완료 시 사라짐

### 모드 B: 미션 모드 (Mission / Objective)
- 미션 조건 제시: "다음 5개 JP 단어를 정확히 입력하라"
- 조건 충족 = 클리어, 실패 = 재시도 또는 페널티
- 격파 메카닉 위에서 동작 (mode A의 변형)

## 핵심 메트릭

| 메트릭 | 정의 | 용도 |
| --- | --- | --- |
| WPM (Words Per Minute) | (정확히 입력된 단어 수) / (분) | 속도 |
| Accuracy | (정확 입력 / 전체 입력) × 100 | 정확성 |
| Score | WPM × Accuracy × 단어 난이도 가중치 | 종합 점수 |
| Combo | 연속 정확 입력 수 | 보너스/콤보 표시 |

## 세션 길이

- 한 스테이지: **3~7분** (10~30개 단어/문장)
- 한 세션: **5~15분** (스테이지 1~3개)
- 메타 진행: 누적 점수, 해금 진척

## 실패 / 재시도

- 스테이지 실패 (시간 초과 또는 HP 0): 스테이지 재시도 또는 다음 스테이지로 스킵 (설정 가능)
- 게임 오버 없음 (RPG 스타일)
- 미션 실패: 스테이지 재시도 또는 부분 보상
