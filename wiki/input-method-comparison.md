# Input Method Comparison

세 언어의 입력 방식을 비교합니다.

## 요약표

| 항목 | EN | JP | ES |
| --- | --- | --- | --- |
| 표시 | 영문자 | 한자/히라가나/가타카나 | 스페인어 (액센트 포함) |
| 입력 | 영문자 (직접) | 로마자 | 스페인어 (Strict) 또는 ASCII (Loose) |
| 길이 비 | 1:1 | 1:N (romaji → 짧은 한자, 긴 한자) | 1:1 또는 1:1.5 (폴백) |
| 난이도 곡선 | 짧은 단어 → 긴 문장 | 히라가나 → 한자 비율 증가 | 액센트 비율 증가 |
| 모드 토글 | 없음 | romaji 힌트 (선택) | Strict/Loose |

## 상세 비교

### EN

- **가장 단순**: 키 입력 → 표시 그대로
- **학습 부담 없음**: 사용자가 모국어라면 즉시 플레이
- **게임 차별점 약함**: 일반 타자 게임과 유사

### JP

- **차별점 강함**: 로마자→한자 매핑이 학습 효과와 재미를 동시에 제공
- **난이도 가파름**: romaji 길이가 한자 길이보다 김 (예: `konnichiwa` 11자 vs `こんにちは` 5자)
- **표기 선택 필요**: 한자/히라가나/가타카나 혼용 시 명확한 표기 권장
- **Pillar 1 핵심**: 이 게임의 가장 강한 차별점

### ES

- **차별점 중간**: 액센트는 게임의 특별한 학습 요소
- **유연성**: Loose 모드로 진입 장벽 낮춤
- **Strict 모드 학습 효과**: 실제 스페인어 입력 학습
- **OS 의존**: Strict 모드는 OS 키보드 환경에 따라 다름 (게임은 무관)

## 입력 길이 비교 (예시)

### 같은 의미의 단어

| 언어 | 단어 | 입력 길이 | 표시 길이 |
| --- | --- | --- | --- |
| EN | hello | 5 | 5 |
| JP | こんにちは | 11 (konnichiwa) | 5 |
| ES | hola | 4 | 4 |

### 난이도 조정

- **EN 후반**: 긴 문장 (40자+)으로 난이도 조정
- **JP 후반**: 한자 비율 + 긴 문장 + romaji 길이 증가
- **ES 후반**: 액센트 + 긴 문장

## 게임 균형

### 적 등장 시간 조정

- JP 적은 평균 1.5~2배 길어 보임 (romaji)
- HP 계산 시: EN 1글자 = JP 1글자 (표시 기준, 입력 기준 아님)
- 점수 계산 시: 단어당 점수 동일 (입력 길이 무관)

### 시각 균형

- EN: 일반 텍스트
- JP: 큰 한자 (시각적 임팩트)
- ES: 액센트 강조 (학습 포인트)
